# 💻 Taskify
뒤죽박죽 할일이 정리가 안되고 있다면?  
좀 더 효율적으로 그룹에서 할일을 분배하고 싶다면?  
'**Taskify**'에서 시작해보세요!  
대시보드를 생성하여 팀원들을 초대하거나 개인적으로 할 일들을 다양한 상태로 관리할 수 있습니다!👍🏻  
담당자와 마감기한을 설정하여 보다 효율적으로 업무 분배 및 생산성을 극대화 해보세요~ 😆
